Portfolio WebApp

Built with Next.js 15, React 19 & Tailwind CSS v4

Tech Stack

Next.js 15 React 19 TypeScript Tailwind CSS v4 shadcn/ui Lucide React

Key Features

App Router

Next.js 15 layout system with server components and static generation.

Dark Mode

System-aware theme toggle with persistent localStorage preferences.

Bento Grid

Responsive dashboard-style layout for showcasing projects and skills.

Markdown Content

Blog and project pages powered by local MDX files with frontmatter.

Static Generation

Optimized builds with generateStaticParams for pre-rendered pages.

shadcn/ui

Accessible, reusable UI primitives built on Radix and class-variance-authority.

Note: This is a static preview of the Portfolio WebApp. The full application includes dynamic routing for projects and blog posts, a theme toggle with dark/light mode, interactive bento grid, and markdown-based content management.

Pages

Home

Hero section, bento grid skills showcase, and latest work highlights.

/projects

Grid of project cards with tag filtering and detail pages.

/blog

Markdown-powered blog engine with tag-based categorization.
