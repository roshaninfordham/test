## What's New in Next.js 15

Next.js 15 brings several exciting improvements to the React framework.

### Async Request APIs

The `cookies()`, `headers()`, and `params` objects are now asynchronous by default, making the data flow more explicit.

### Improved Static Generation

Better static generation with enhanced caching strategies for faster builds and deployments.

### React 19 Support

Full support for React 19's new features including the new hook APIs and improved server component architecture.
