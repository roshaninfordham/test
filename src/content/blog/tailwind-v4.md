## Tailwind CSS v4

Tailwind CSS v4 represents a fundamental shift in how Tailwind is configured.

### CSS-First Configuration

Gone is the JavaScript config file. All customization happens in your CSS file using `@theme` directives.

### Performance

The new engine is built on Lightning CSS, providing dramatically faster build times.

### New Features

- CSS-first configuration with `@theme`
- Improved dark mode with `@variant dark`
- Better composability with `@reference`
