## Overview

A collaborative task management tool inspired by Trello and Linear, built for teams.

## Features

- **Kanban Boards** with drag-and-drop
- **Real-time collaboration** via WebSockets
- **AI task suggestions** using natural language processing
- **Role-based access control**
- **Rich text descriptions** with markdown support
