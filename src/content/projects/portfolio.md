## Overview

This portfolio website showcases projects, skills, and blog posts using a clean, modern design. Built with the latest web technologies.

## Features

- **Next.js 15 App Router** with static site generation
- **React 19** with server and client components
- **Tailwind CSS v4** for utility-first styling
- **Dark/Light mode** with system preference detection
- **Markdown-based content** for easy updates
- **Responsive design** for all devices
- **Bento Grid** layout for project showcase

## Architecture

The application uses a hybrid architecture with server components for content rendering and client components for interactive features like theme toggling.
